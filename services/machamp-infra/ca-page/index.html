Installation Instructions

  1. Tap Download Certificate above
  2. Go to Settings → General → VPN & Device Management
  3. Tap the downloaded profile and install it
  4. Go to Settings → General → About → Certificate Trust Settings
  5. Toggle on Team Rocket under Enable Full Trust

  1. Download the certificate
  2. Go to Settings → Security → Encryption & Credentials
  3. Tap Install a certificate → CA certificate
  4. Select the downloaded file

  1. Open the downloaded homelab-ca.crt
  2. Keychain Access opens — add to System keychain
  3. Find Team Rocket in the list and double-click it
  4. Expand Trust → set SSL to Always Trust
  5. Close and enter your password to confirm

  1. Double-click the downloaded homelab-ca.crt
  2. Click Install Certificate…
  3. Select Local Machine, click Next
  4. Choose Place all certificates in the following store → Trusted Root Certification Authorities
  5. Click Finish and accept the security warning

  1. Copy the cert: sudo cp homelab-ca.crt /usr/local/share/ca-certificates/
  2. Update the store: sudo update-ca-certificates
  3. For Chrome/Firefox, also import via browser certificate settings
After installation, reload any open .home tabs. If you still see a warning, ensure you enabled full trust (iOS) or installed to the correct keychain/store.
